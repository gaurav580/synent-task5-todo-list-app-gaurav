# To-Do List Web Application

## Objective
Develop a responsive To-Do List application using HTML, CSS, and JavaScript.

## Features
- Add Tasks
- Delete Tasks
- Mark Tasks as Completed
- Local Storage Support
- Responsive User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
Download the project and open index.html in your web browser.
